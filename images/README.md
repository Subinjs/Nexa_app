# Add Model Photos

To display actual car images in the features page, add photos to the `images/` folder with these names:

## Required Images for Each Model

### Baleno
- `baleno_front.jpg` - Front view
- `baleno_side.jpg` - Side profile
- `baleno_rear.jpg` - Rear view
- `baleno_interior.jpg` - Interior/dashboard

## Image Specifications

- **Recommended size**: 800x600px or higher
- **Format**: JPG or PNG
- **Max file size**: 500KB per image (for faster loading)
- **Aspect ratio**: 4:3 or 16:9

## After Adding Images

The features page will automatically detect and display the images. Currently showing placeholder boxes with image names.

## Optional: Update features.html

If you want different image formats or paths, edit the `displayFeatures()` function in [features.html](features.html).
